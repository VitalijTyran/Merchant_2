2`Проект Merchant
-----------------

В проекте производится расчет
заработной платы продавца, в зависимости
от объема продаж, по прогрессивной шкале.


ЗАДАНИЕ:
1) На примере проекта Merchant создать проект Merchant-01.
2) Разработку Merchant-01 вести под Git и на GitHub.
3) В Merchant-01 должны быть ветки master и develop.
4) Функционал ветки master идентичен функционалу Merchant.
5) В ветке develop реализовать:
 - дополнительно ввод фамилии продавца.
 - изменение шкалы вознаграждения продавца с 3-х уровневой
 на 4-х.
6) При разработке отработать: коммиты, ветвление, слияние.
7) Слить ветку develop с веткой master, оставив приоритетом
функционал ветки develop. После слияния с веткой master,
ветку develop не удалять.
8) После завершения проекта в LMS указать ссылку на GitHub.
